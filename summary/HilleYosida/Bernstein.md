## Basic properties of CM functions

### `IsCompletelyMonotone` — Definition

**Lean signature**
```lean
def IsCompletelyMonotone (f : ℝ → ℝ) : Prop
```

**Informal**: A function $f : \mathbb{R} \to \mathbb{R}$ is *completely monotone* on $[0,\infty)$ if it is $C^\infty$ on $[0,\infty)$ and $(-1)^n f^{(n)}(t) \geq 0$ for all $n \in \mathbb{N}$, $t \geq 0$. This is Widder's definition ("The Laplace Transform", p. 145).

## Bernstein's Theorem

### [`bernstein_theorem`](../../HilleYosida/Bernstein.lean#L91) — Theorem

**Statement**: If $f$ is completely monotone, then there exists a finite positive measure $\mu$ supported on $[0, \infty)$ such that

$$f(t) = \int_0^\infty e^{-tp}\, d\mu(p) \quad \text{for all } t \geq 0.$$

**Proof uses**: `Nat.factorial_pos`, `Nat.cast_pos`, and *1 sorry* (Phases 2–5: Taylor remainder, pushforward, Prokhorov, Portmanteau). The density $\rho_n(t) = \frac{(-1)^n}{(n-1)!} t^{n-1} f^{(n)}(t)$ is defined and its nonnegativity is fully proved.
