## What was done

### Proofs filled (8 → 4 sorries)

1. **`normBoundedOnUnitInterval`** — Uniform operator norm bound on [0,1] via
   Banach-Steinhaus. Key insight: for each x, the orbit {S(t)x : t ∈ [0,1]}
   is bounded (iterate semigroup from a δ-neighborhood where strong continuity
   gives a bound), then apply `banach_steinhaus` from
   `Mathlib.Analysis.Normed.Operator.BanachSteinhaus`.

2. **`existsGrowthBound`** — Every C₀-semigroup has exponential growth M·e^{ωt}.
   Decompose t = ⌊t⌋₊ + frac, bound ‖S(k)‖ ≤ M^k by induction on integer
   times, then M^n ≤ exp(log M · t) since n ≤ t.

3. **`integrable_resolvent_integrand`** — Integrability of e^{-λt}·S(t)x on (0,∞).
   Two parts: (a) norm bound ‖e^{-λt}·S(t)x‖ ≤ e^{-λt}·‖x‖ using contraction,
   then `Integrable.mono'` with `exp_neg_integrableOn_Ioi`; (b) measurability
   via `ContinuousOn.aestronglyMeasurable` using `strongContAt`.

4. **Resolvent norm bound** — ‖R(λ)x‖ ≤ (1/λ)·‖x‖. Uses
   `norm_integral_le_of_norm_le` for step 1, then evaluates ∫ exp(-λt) = 1/λ
   via `integral_comp_mul_left_Ioi` (substitution u=λt) + `integral_exp_neg_Ioi_zero`.

5. **`hilleYosidaResolventBound`** — ‖R(λ)‖ ≤ 1/λ. One-liner:
   `LinearMap.mkContinuous_norm_le _ (by positivity) _`.

### Resolvent refactoring

Moved resolvent from `StronglyContinuousSemigroup` to `ContractingSemigroup`:
- **Why**: General C₀-semigroups can grow as e^{ωt}; if λ ≤ ω, the Bochner
  integral diverges and returns junk (0), breaking `integral_add`.
- **How**: `LinearMap.mkContinuous` with bound 1/λ. Proved `map_add'` via
  `integral_add` + integrability, `map_smul'` via `smul_comm` + `integral_smul`.
- Added `SMulCommClass ℝ ℝ X` instance (needed at `maxSynthPendingDepth = 3`).
